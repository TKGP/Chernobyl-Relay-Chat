Встроенный чат-клиент для игры "Call of Chernobyl" 1.4.12 или более поздней версии, с любым аддоном.
Сообщения об ошибках и предложения отправлять на GitHub или по электронной почте: contact.tkgp@gmail.com


--| Установка

1. Установить .NET framework (если не установлен): https://www.microsoft.com/net/download/framework
2. Скопировать папку gamedata в директорию с игрой


--| Использование

Запустить Chernobyl Relay Chat.exe; приложение должно быть запущено, чтобы работал игровой чат, вы также можете использовать его в качестве отдельного клиента.
После подключения, нажмите кнопку ОПЦИИ, чтобы изменить свое имя и другие настройки, а затем запустить СОС.
После игры, нажмите Enter (по умолчанию), чтобы открыть интерфейс чата и снова Enter, чтобы отправить сообщение, или Escape для выхода без отправки сообщения.
Вы можете использовать текстовые команды из игры или клиента, начиная с /. Чтобы увидеть все доступные команды, используйте /.
